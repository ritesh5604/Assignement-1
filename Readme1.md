# Weather Wizard 🌤️

## 🛠 Tech Stack
- React (via Vite)
- Material UI (MUI)
- Framer Motion
- OpenWeatherMap API
- Vercel for deployment

## ⚙️ Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` with your API key:
4. Run `npm run dev` to start the app

## 🌐 API Info
- Base URL: `https://api.openweathermap.org/data/2.5/`
- Rate limit: 60 requests/min (free tier)
- API key required
